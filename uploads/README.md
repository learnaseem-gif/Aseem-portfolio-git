# Drop your media here

Upload your real photos and reels into **this `uploads/` folder** on GitHub.
I'll then optimise them, move them into `assets/work/<client>/`, and wire them
into the site — you don't need to touch any code.

## How to upload from your phone

1. Open **github.com/learnaseem-gif/aseem-portfolio-git** in a browser
2. Switch to the branch **`claude/impeccable-install-setup-paaeel`** (branch dropdown, top-left of the file list)
3. Open this `uploads/` folder → **Add file → Upload files**
4. Select your photos/videos, then **Commit changes**

## Naming (so I can auto-sort them)

Name each file starting with the client, like this:

| Client | Cover image | More photos | Reel / video |
|---|---|---|---|
| Porsche Centre Richmond | `porsche-cover.jpg` | `porsche-1.jpg`, `porsche-2.jpg` … | `porsche-reel.mp4` |
| Bagga Jewels | `bagga-cover.jpg` | `bagga-1.jpg`, `bagga-2.jpg` … | `bagga-reel.mp4` |
| Saloud | `saloud-cover.jpg` | `saloud-1.jpg` … | `saloud-reel.mp4` |
| Canwide Mortgage | `canwide-cover.jpg` | `canwide-1.jpg` … | `canwide-reel.mp4` |
| The Motor World Auto's | `motorworld-cover.jpg` | `motorworld-1.jpg` … | `motorworld-reel.mp4` |

- **cover** = the main image for the card + top of the project page
- **numbered** = extra photos for that project's gallery
- **reel** = a short video that plays as the hero on the project page

Don't stress about exact names — if they're roughly like this I'll sort it out.
Just tell me which file belongs to which client if it's ambiguous.

## Formats & size

- **Photos:** JPG or PNG. Any size is fine — I'll compress them.
- **Video:** MP4 (H.264) works best. Please keep each **under ~50 MB**
  (GitHub's hard limit is 100 MB per file). If a reel is bigger, send the
  shorter/compressed version — we can host full videos elsewhere later.

Once you've uploaded, just say "done" and I'll take it from here.
